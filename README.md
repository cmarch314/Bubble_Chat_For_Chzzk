# 🎮 BubbleChat 치트시트

치지직 채팅과 연동되는 **인터랙티브 방송 오버레이**입니다.  
채팅에 따라 **말풍선**, **효과음**, **화면 연출**이 자동으로 작동합니다.

---

## 🚀 빠른 시작 (Quick Start)

### 1. 설정 및 실행
1. `index.html` 파일을 텍스트 에디터로 엽니다.
2. `<div id="id" chzzkHash="...">` 부분에 본인의 **치지직 채널 고유 ID(Hash)**를 입력합니다.
   - *채널 URL의 마지막 부분 (예: `chzzk.naver.com/live/THIS_PART`)*
3. OBS 등 방송 송출 프로그램의 '브라우저 소스'로 `index.html`을 추가하거나, 파일을 직접 웹브라우저로 엽니다.

### 2. 관리자 명령어 (채팅창 입력)
- **`!데모 [초]`**: 설정한 시간(기본 60초) 동안 랜덤 채팅 시뮬레이션을 실행합니다. (예: `!데모`, `!데모 120`)
- **`!데모 끝`**: 데모 시뮬레이션을 즉시 중단합니다.
- **`!사운드 [on/off]`**: 전체 효과음(SFX)을 켜거나 끕니다. (예: `!사운드 off`)
- **`!비주얼 [on/off]`**: 전체 화면 이펙트를 켜거나 끕니다. (예: `!비주얼 off`)
- **`!볼륨평준화 [on/off]`**: 큰 소리를 자동으로 줄여주는 [오디오 컴프레서] 기능을 제어합니다.

## 🎨 특수 연출 (Visual Effects)

채팅으로 특정 키워드를 입력하거나 도네이션 시 화려한 연출이 나옵니다.

| 명령어 | 설명 | 비고 |
| :--- | :--- | :--- |
| **`!발파 [메시지]`** | **[New]** 발파루크(제트기+유성) 시네마틱 연출 | 18초 지속, 웅장함 |
| **`!해골 [메시지]`** | 화면이 어두워지며 해골이 등장하는 공포 연출 | |
| **`!우쇼`** | 우쇼 댄스 GIF와 함께 사이키델릭한 조명 | 이미지 파일 필요 |
| **`!커플`** | 두 이모지가 합쳐지는 로맨틱한 연출 | |
| **`!하트`** | 화면 가득 하트가 쏟아지는 사랑스러운 연출 | |
| **`!버질`** | 화면을 베는 듯한 참격 연출 (Vergil) | |
| **`돌핀`** | (구독 시 자동) 화면 하단에 물결과 점프하는 돌핀 | 자동 발동 |

## ⚙️ 상세 설정 (config.js)

`config.js` 파일을 열어 기능을 튜닝할 수 있습니다.

### 1. 오디오 설정 (`HIVE_VOLUME_CONFIG`)
- **`master`**: 전체 볼륨 (0.0 ~ 1.0)
- **`visual`**: 특수 연출 사운드 볼륨
- **`sfx`**: 일반 채팅 효과음 볼륨

### 2. 고급 오디오 (`NORMALIZER_CONFIG`)
- **`enabled`**: `true`로 설정 시 소리 평준화 기능 활성화
- 급격히 큰 소리가 나도 귀가 아프지 않게 자동으로 조절해줍니다.

### 3. 비주얼 튜닝 (`VISUAL_CONFIG`)
각 이펙트의 지속 시간, 이모지 개수, 속도 등을 섬세하게 조절할 수 있습니다.
- **`valstrax`**: 제트기 딜레이, 유성 충돌 시간 등
- **`dolphin`**: `creatureSize`로 바다 생물 크기 조절
- **`heart`**: 떨어지는 하트의 회전 각도, 텍스트 분할 방식 등
