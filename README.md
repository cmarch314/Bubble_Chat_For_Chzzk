# 🎮 BubbleChat 치트시트

치지직 채팅과 연동되는 **인터랙티브 방송 오버레이**입니다.  
시청자가 채팅창에 특정 단어나 명령어를 입력하면 **말풍선**, **효과음(SFX)**, **반응형 비디오**, **화면 연출 이펙트**가 실시간으로 작동합니다.


### 🎬 1. 오버레이 명령어 리스트 (Commands List)

#### 📢 후원 및 연출 이펙트 (Screen Effects)

## 🛠️ 2. 스트리머를 위한 안내 (Streamer Guide)

### 🚀 설정 및 실행
1. `index.html` 파일을 텍스트 에디터로 엽니다.
2. `<div id="id" chzzkHash="...">` 부분에 본인의 **치지직 채널 고유 ID(Hash)**를 입력합니다.
3. OBS의 '브라우저 소스'로 추가하거나 웹브라우저로 직접 엽니다.

### 🎮 관리자 명령어 (채팅창 전용)
- **`!데모 [초]`**: 테스트용 랜덤 채팅 시뮬레이션을 실행합니다.
- **`!데모 끝`**: 시뮬레이션을 즉시 중단합니다.
- **`!사운드 [on/off]`**: 전체 효과음(SFX)을 제어합니다. (`!소리끄기`, `!소리켜기`, `!음소거` 포함)
- **`!비주얼 [on/off]`**: 전체 화면 이펙트를 제어합니다.
- **`!볼륨평준화 [on/off]`**: 오디오 컴프레서 기능을 켜거나 끕니다.

### ⚙️ 상세 설정 (config.js)
`config.js` 파일에서 더 세밀한 조정이 가능합니다.
- **`HIVE_VOLUME_CONFIG`**: 마스터, 비주얼, SFX 각 영역별 볼륨 조절.
- **`VISUAL_CONFIG`**: 각 이펙트의 지속 시간, 캐릭터 크기, 이동 영역 조정.
- **`NORMALIZER_CONFIG`**: 소리 평준화(컴프레서) 기능 활성화 여부.
