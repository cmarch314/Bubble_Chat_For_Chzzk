# 🎮 BubbleChat 치트시트

## ⚙️ 설정 가이드 (Configuration)
`config.js` 파일을 통해 다음 기능들을 설정할 수 있습니다:
- **채널 ID**: `window.CHZZK_CHANNEL_ID` 혹은 `script.js`의 로직을 통해 채널을 설정합니다.
- **환영 메시지**: `window.WELCOME_MESSAGES`에 방송 접속 시 출력될 랜덤 메시지를 추가할 수 있습니다.
